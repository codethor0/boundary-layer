# Demo Assets

BoundaryLayer ships a **sanitized demo transcript** captured from real `make smoke` and `make demo` runs. Terminal GIFs are optional and require local tooling.

## Generated artifacts

| Asset | Path | Status |
|-------|------|--------|
| Demo transcript | [docs/assets/demo-transcript.txt](assets/demo-transcript.txt) | Generated by `make capture-demo` |
| asciinema cast | `docs/assets/boundary-layer-demo.cast` | Only if `asciinema` is installed |
| Terminal GIF | `docs/assets/boundary-layer-demo.gif` | Only if `asciinema` and `agg` are installed |

GIF not generated unless capture tooling is available. The transcript is the default reproducible artifact.

## Regenerate transcript

With the dev stack running:

```bash
make up
make capture-demo
```

Output is sanitized (home paths and secret-like patterns redacted).

## Convert transcript or cast to GIF later

If you record a cast:

```bash
asciinema rec docs/assets/boundary-layer-demo.cast
make smoke
make demo
# exit recording
agg docs/assets/boundary-layer-demo.cast docs/assets/boundary-layer-demo.gif
```

Or share the transcript excerpt in README/workshop slides without a GIF.

## Recommended README screenshots

When captured, place PNG files under `docs/assets/` and link from README. Do not add placeholder images.

## Exact commands for alert webhook sample

```bash
curl -sf -X DELETE http://localhost:8081/alerts
curl -sf -X POST http://localhost:8000/labs/circuit-breaker/run \
  -H "Content-Type: application/json" -d '{"mode":"hardened"}'
sleep 30
curl -sf http://localhost:8081/alerts | python3 -m json.tool
```

Expected alert name: `BoundaryLayerInferenceCircuitBreakerOpen`.
